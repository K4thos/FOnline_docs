Определяет максимальную дистанцию, с которой можно инициировать диалог с NPC.

Прим.
Если значение стата равно 0, то тогда используется значение из глобальной переменной __TalkDistance.
См. также
__TalkDistance
#define ST_SCALE_FACTOR   (116)
#define ST_MAX_TALKERS   (119)

Максимальное количество одновременно запущенных диалогов.

ST_MAX_TALKERS < 0 - диалог не может быть начат; ST_MAX_TALKERS == 0 - значение берется из глобальной переменной __NpcMaxTalkers; ST_MAX_TALKERS > 0 - берется данное значение.
