Типы персонажей (CritterTypes.cfg)

Типы персонажей задаются в файле Data\CritterTypes.cfg, находящемся в папке сервера.

Строки, начинающиеся с символа "#", являются комментариями и не влияют на работу сервера, символ "@" является признаком данных о персонаже, после символа "@" идет перечисление параметров типа персонажей, разделенные одним или несколькими пробелами.

Номер типа 0-999
Имя Name
Номер алиаса (используется для выбора изображения для выбора части тела при прицельном выстреле) Alias
Параметр мультигексовости (0 - тип персонажа не мультигексовый) MH
Используется ли 3D-модель 3d
Может ли персонаж ходить Walk
Может ли персонаж бегать Run
Может ли персонаж прицельно атаковать Aim
Может ли персонаж носить броню Armor
Может ли персонаж вращаться Rotate
Имеется ли стандартный набор анимаций без оружия A (Empty)
Имеется ли анимация падения и смерти B (KO/Dead)
Имеется ли анимация вставания после падения C (KO Rise)
Имеется ли анимация атаки ножом D (Knife)

Количество миллисекунд, которое потребуется криттеру для перехода на другой гекс бегом Run
Кадр анимации, завершающий переход на один гекс Walk steps: значение 1
Кадр анимации, завершающий переход на два гекса Walk steps: значение 2
Кадр анимации, завершающий переход на три гекса Walk steps: значение 3
Кадр анимации, завершающий переход на четыре гекса Walk steps: значение 4
