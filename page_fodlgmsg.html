FODLG.MSG

В файле FODLG.MSG содержится описание NPC по умолчанию, то есть тех, кому не присвоен свой уникальный диалог с описаниями.

Формат:

{PID NPC*10}{}{Имя/название}
{(PID NPC*10)+1}{}{Расширенное описание}

Пример:

{90}{}{Пес}
{91}{}{Собака, наверняка бездомная, дворняжьей породы.}

Такое описание будет у всех NPC с PID=9.

Обычно, для присваивания нового описания рекомендуется просто присвоить NPC свой диалог, чтобы не создавать много лишних прототипов с одинаковыми характеристиками. Однако, даже если в файле диалога нет никаких фраз, то во всплывающем меню, появляющемся при нажатии на NPC, все равно первой опцией будет "начать диалог" (по умолчанию - бинокль). А это может мешать в бою - при клике по NPC с целью посмотреть параметры персонаж пытается подойти и поговорить с ним. Поэтому, если нужны какие-то NPC без диалогов, которые активно будут использоваться в боевых действиях, лучше создать им новые прототипы со своим описанием для удобства игрока.
