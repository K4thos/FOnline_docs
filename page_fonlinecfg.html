FOnline.cfg

Настройки клиента.

Ключи:

Language Язык клиента по умолчанию
RemoteHost Домен или ip сервера игры
RemotePort Порт сервера
ProxyType Тип прокси (0 - Нет прокси, 1 - Socks4, 2 - Socks5, 3 - HTTP)
ProxyHost Адрес прокси-сервера
ProxyPort Порт прокси-сервера
ProxyUser Имя пользователя для прокси
ProxyPass Пароль для прокси
DefaultCombatMode Режим боя по умолчанию (используется в скриптах сервера при выборе режима боя, варианты: 0 - в реальном времени и пошаговом, 1 - только в реальном, 2 - только в пошаговом)
MusicVolume Громкость музыки от 0 до 100
SoundVolume Громкость звуков от 0 до 100
GlobalSound Будут ли воспроизводится звуки, когда игра не активна. (1 - будут, 0 - не будут)
ScreenWidth Ширина окна клиента
ScreenHeight Высота окна клиента
FullScreen Запускать ли клиент в полноэкранном режиме (1 - запускать, 0 - не запускать)
AlwaysOnTop Показывать ли окно клиента в режиме "поверх всех окон" (1 - включить, 0 - отключить)
AlwaysRun Опция постоянного бега, по умолчанию выключена
Light Общий уровень освещенности
ScrollDelay Плавность скроллинга (от 1 до 32)
ScrollStep Дополнительное уменьшение скорости скроллинга (4, 8, 16, 32).
MouseSpeed Скорость движения мышки в полноэкранном режиме
TextDelay Базовое время, в течении которого показывается текст реплик. В миллисекундах (Расчет: TextDelay + 1 символ * 100 мс.)
DamageHitDelay Время отображения информации о повреждении. В миллисекундах (0 - выключено. Рекомендуемое значение при включенном состоянии - 2000)
Sleep Уменьшение нагрузки на процессор. Чем больше значение, тем меньше загрузка процессора и FPS. От -1 (выкл) до 10.
LangChange Комбинация для переключения раскладки в чате. 0 - Ctrl+Shift, 1 - Alt+Shift, 2-Right Ctrl
IndicatorType Индикация патронов и износа (0 - полоски, 1 - цифры, 2 - полоски и цифры)
InvertMessBox Инвертирование окна сообщений (0 - сообщения идут снизу вверх (новые снизу, как в оригинале), 1 - сообщения идут сверху вниз (новые сверху))
WinNotify Если при неактивном окне клиента вам приходит сообщение, то в панеле задач моргнет иконка. Это не относится к сообщениям о результатах атаки и некоторым другим сообщениям (0 - выключено, 1 - включено (default))
SoundNotify Если при неактивном окне клиента вам приходит сообщение, то прозвучит звуковой сигнал. Это не относится к сообщениям о результатах атаки и некоторым другим сообщениям (0 - выключено (default), 1 - включено)
DoubleClickTime Время ожидания второго клика для бега, в миллисекундах. При нуле настройки берутся из настроек ОС
CombatMessagesType Тип боевых сообщений (0 - полное/verbose, 1 - краткое/brief)
FonlineDataPath Путь к основной папке ресурсов FOnline
MasterDatPath Путь к файлу master.dat
CritterDatPath Путь к файлу critter.dat
FlushValue Кэшируемые спрайты
BaseTexture Размер текстуры, где будут храниться спрайты. Должен быть степенью двойки - 256, 512, 1024...
VSync Вертикальная синхронизация (1 - включено, 0 - отключено)
BackGroundClear Дополнительная очистка экрана (1 - включено, 0 - отключено)
MultiSampling Сглаживание 3d (antialiasing) (-1 - автоопределение, 0 - выключено, 1..16 - значение сглаживания)
SoftwareSkinning Програмный скиннинг (1 - включено, 0 - отключено)
Animation3dFPS Имитация покадровой графики 3d моделями. Количество кадров в секунду. При 0 - выключено, анимация будет воспроизводиться максимально плавно; при включении оптимальное значение - 10.
Animation3dSmoothTime Сглаживание переходов в 3d графике, время в миллисекундах. Актуально только при Animation3dFPS=0.
Logging Вести ли лог (1 - включено, 0 - отключено)
LoggingDebugOutput Вести ли лог отладочных сообщений (1 - включено, 0 - отключено)
LoggingTime Добавлять ли время в лог (1 - включено, 0 - отключено)
LoggingThread Добавлять ли метку потока в лог (1 - включено, 0 - отключено)
ServerAppName Имя файла сервера
ServerPath Путь к папке сервера
ServerCommandLine Дополнительные параметры в коммандной строке сервера
RemoteHost_0 Ранее используемый адрес сервера. Используется только конфигнуратором клиента FOnline для удобства выбора сервера из выпадающего списка. По аналогии могут существовать ключи RemoteHost_1, RemoteHost_2, RemoteHost_3 и т.д.
KeyboardRemap Возможность перенаправления ввода кодов символов клавиатуры. Производится через пробел - X Y X Y..., где X - код, который меняем, Y - на который. Номера кодов - http://fonline.pastebin.com/Qy3E0qwj
