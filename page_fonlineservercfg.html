FOnlineServer.cfg

Настройки сервера.

Ключи:

PositionX Положение окна сервера по горизонтали. 0 - центр экрана
PositionY Положение окна сервера по вертикали. 0 - центр экрана
TextFont Шрифт
Port Порт сервера
GameSleep Время простоя после выполнения цикла. В миллисекундах
WorldSaveTime Промежуток времени между сохранениями игрового мира. В минутах
WorldSaveManager Режим сохранения мира. 0 - напрямую на диск, 1 - через отдельный поток, в котором так же будет происходить удаление старых сохранений
LogicThreadCount Количество потоков в игровой логике. При нуле количество потоков будет равно количеству процессоров на компьютере
LogicThreadSetAffinity Количество потоков на 1 ядро, если 0, то операционная система автоматически распределит потоки между ядрами
ScriptConcurrentExecution Возможности одновременного выполнения скриптов (0 - отключено)
NetWorkThread Количество сетевых потоков, если 0, то система определит количество потоков по количеству процессоров в системе
MemoryDebugLevel Мониторинг использования памяти (0 - отключено, 1 - простой мониторинг, 2 - глубокий мониторинг)
Logging Вести ли лог (1 - включено, 0 - отключено)
LoggingDebugOutput Вести ли лог отладочных сообщений (1 - включено, 0 - отключено)
LoggingTime Добавлять ли время в лог (1 - включено, 0 - отключено)
LoggingThread Добавлять ли метку потока в лог (1 - включено, 0 - отключено)
Language_0 Язык сервера (для мультиязычных серверов добавляются еще ключи Language_1, Language_2, Language_3 и т.д. по количеству языков)
Access_client Пароли, необходимые для смены уровня доступа на Client (пароли разделяются пробелом)
Access_tester Пароли, необходимые для смены уровня доступа на Tester (пароли разделяются пробелом)
Access_moder Пароли, необходимые для смены уровня доступа на Moderator (пароли разделяются пробелом)
Access_admin Пароли, необходимые для смены уровня доступа на Administrator (пароли разделяются пробелом)
