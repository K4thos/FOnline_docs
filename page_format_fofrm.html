Спецификация .fofrm

fps

Frame Per Second, количество кадров в секунду. Целое, беззнаковое. Не обязательный параметр. По умолчанию - 10.

count

Количество кадров в анимации. Целое, беззнаковое. По умолчанию - 1.

offs_x
offs_y

Базовые смещения. Этот сдвиг нужен для того, чтобы кадры были правильно отцентрованы. Используется при отображении кадра, так как его логический центр не всегда совпадает с его абсолютным центром (width/2, height/2). Например, центром существа будет позиция его ног, в то время как центром дерева будет его основание. Положительное значение для offs_x означает, что кадр при отображении будет сдвинут вправо. Положительное значение для offs_y означает, что кадр при отображении будет сдвинут вниз. Целое, знаковое. Не обязательный параметр. По умолчанию - 0.

[dir_?]

Перед блоками описаний кадров анимации ставится направление анимации. ? - это направление, в диапазоне от 0 до 5. В кадрах без учета направления это можно не указывать. Далее следуют элементы блока описаний анимации.

offs_x
offs_y

Смещения для направления. Если они не заданы, то берутся глобальные базовые смещения. Целое, знаковое. Не обязательный параметр. По умолчанию - значение базовых глобальных offs_*.

next_x_?
next_y_?

Смещение кадра относительно предыдущего кадра. ? означает номер кадра*. Целое, знаковое. Не обязательный параметр. По умолчанию - 0.

frm_?

Имя файла изображения. ? означает номер кадра*. Для первого кадра номер указывать не обязательно, то есть просто "frm=...". Должен включать путь от текущей папки. Строка. Обязательный параметр.

Номера должны начинаться с 0 и идти по порядку.
