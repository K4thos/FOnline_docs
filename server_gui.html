GUI сервера
  1. Область лога
    • В данной области отображается служебная информация о работе сервера и запущенных скриптах, необходима для отслеживания правильности функционирования сервера.
  2. Краткая статистика работы сервера
    • Игровое время и ускорение времени сервера
    • Количество соединений с сервером
    • Количество игроков находящихся в игре (в том числе и те игроки, с кем было разорвано соединение, но не вышедшие из игры)
    • Количество NPC
    • Количество локаций и карт
    • Количество предметов
    • Количество игровых переменных
    • Количество сохраненных массивов AnyData
    • Количество установленных временных событий
    • Количество циклов выполняемых в секунду (сервер за один цикл обрабатывает только 1 запрос от клиента)
    • Время цикла
    • Объем отправленных сервером данных
    • Объем принятых сервером данных
    • Среднее значение уровня сжатия.
  3. Кнопки управления
    • Отправляет подключенным клиентам клиентские скрипты (может понадобится при обновлении скриптов клиента)
    • Сохраняет игровой мир игнорируя стандартное расписание
    • Сохраняет лог сервера в файл
    • Сохраняет в файл содержимое области статистики
    • Сохраняет дамп памяти
  4. Кнопки статистики
    • Статистика использования памяти
    • Статистика Игроков, находящихся на сервере в данный момент
    • Информация о картах и локациях
    • Сохраненные временные события
    • Показывает сохраненные на сервере массивы AnyData
    • Статистика по предметам в игре.
  5. Дополнительные опции для областей лога и статистики
    • При включенном значении область статистики (7) обновляется каждую секунду
    • Включает ведение лога
    • Включает ведение лога со временем
    • Включает ведение лога с потоками
    • Вывод в лог сообщений скриптов (при использовании в скриптах функции void Log(string)).
  6. Кнопка запуска и остановки сервера.
  7. Область статистики.
